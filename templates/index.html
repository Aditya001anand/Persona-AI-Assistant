About Persona

Persona is an interactive web application designed to showcase the power and creativity of large language models through advanced prompt engineering.

This project is powered by Google's cutting-edge Gemini AI. Each personality you see has a unique set of instructions that guides the AI's tone, style, and responses, demonstrating how context can create vastly different conversational experiences.

A Project for Prompt Engineering by Aditya Anand
